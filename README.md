# projeto do site de livraria 

Descrição do Projeto – Plataforma de Livros Online

Este projeto consiste no desenvolvimento de uma plataforma completa de venda de livros, criada como trabalho acadêmico por uma equipe de três integrantes. O objetivo foi construir um sistema funcional, moderno e responsivo, utilizando tecnologias amplamente utilizadas no mercado.

A aplicação foi desenvolvida com React, JavaScript, PHP, Tailwind CSS e integrada a um banco de dados, garantindo uma experiência fluida e estruturada para o usuário. O sistema possui fluxo completo, desde o cadastro até a finalização da compra.

Entre as principais funcionalidades, destacam-se:

Cadastro de Usuários: formulário completo com armazenamento dos dados no banco.

Tela de Login: autenticação segura, permitindo acesso às áreas restritas.

Home com Cards de Livros: exibição de livros com imagens, títulos e descrições, tudo carregado de forma dinâmica.

Sistema de Carrinho de Compras: adição, remoção e resumo dos itens selecionados.

Tela de Pagamento: etapa final da compra, integrando todas as informações necessárias.

Navegação via Rotas (React Router): proporcionando uma experiência fluida entre todas as páginas.

Uso de Bibliotecas do React: para otimização de componentes e melhoria da interface.

Interface estilizada com Tailwind CSS: garantindo um layout moderno, responsivo e consistente.

O resultado é um sistema completo e funcional, que demonstra domínio das tecnologias utilizadas, trabalho em equipe e capacidade de entregar uma solução real e escalável.
